---
title: 'Create Database'
api: 'POST /create-database'
description: 'Create a new database for file indexing'
---

This endpoint is used to create a new database where you can index files.

### Parameters

<ParamField body="databaseName" type="string" required={false}>
  Optional. Name for your database. Must be unique within your account.
</ParamField>

<ParamField body="apiKey" type="string" required={true}>
  Your Captain API key.
</ParamField>

<ParamField body="clientID" type="string" required={true}>
  Your Captain user ID.
</ParamField>

### Response

<ResponseField name="databaseName" type="string">
  Confirmation of the created database name.
</ResponseField>

<ResponseField name="success" type="boolean">
  Indicates if the database creation was successful.
</ResponseField>

<ResponseField name="error" type="string">
  Error message if database creation failed.
</ResponseField>

### Error Codes

<ResponseField name="400" type="object">
  Unsupported databaseName strings.
</ResponseField>

<ResponseField name="409" type="object">
  The databaseName is already in use.
</ResponseField>

### Example Request

```bash
curl -X POST https://api.runcaptain.com/create-database \
  -H "Content-Type: application/json" \
  -d '{
    "databaseName": "my-knowledge-base",
    "apiKey": "your_api_key",
    "clientID": "your_client_id"
  }'
```

### Example Response

```json
{
  "databaseName": "my-knowledge-base",
  "success": true
}
```
